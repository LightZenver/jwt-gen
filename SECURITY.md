## Security Notice

This is a JWT key generator for TESTING and DEVELOPMENT purposes only.

⚠️ **NEVER use generated keys in production**
⚠️ **Generated keys are for testing only**
⚠️ **Keep secrets secure and never commit them**

## Reporting a Vulnerability

If you discover a security vulnerability, please DO NOT create a public issue.
Instead, contact the maintainers directly.
